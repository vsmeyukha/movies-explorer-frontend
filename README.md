# Frontend-часть SPA-приложениями по поиску фильмов. 

Функционал: 
- запросы к API;
- фильтрация базы фильмов по введённому пользователем запросу;
- Фильтрация по короткометражкам;
- Выбор цветовой темы;

Стек: HTML, CSS, Flexbox, Grid, React.
